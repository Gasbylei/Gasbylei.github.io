Laravel Collection常用方法

发表于 2017-06-05 |

自从Laravel 5.3开始，关于Eloquent ORM操作的返回值变成了Collection，不再是数组，意味着我们对Collection的操作将会变得越来越多，下面总结了一些Collection的常用方法，方便以后使用。

简介

Illuminate\Support\Collection类提供了一个流畅、便利的封装来操控数组数据。

这个有三个方法值得你去看看：

  • whereIn() - 基于参数中的键值数组进行过滤
  • search() - 在集合内搜索指定的数值并返回找到的键。假如找不到项目，则返回 false
  • has() - 检查集合中是否含有指定的「键」,返回布尔值

数据排序

关于集合的排序很简单，集合提供了两种方法:

  • sortBy() - 根据一定标准进行排升序
  • sortByDesc() - 根据一定标准排降序

sort()方法可以使用给定键值或回调函数进行排序

values()方法来把键重置为连续数字的键,toArray()方法将集合转换成数组。
