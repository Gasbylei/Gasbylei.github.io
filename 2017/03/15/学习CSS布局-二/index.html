使用inline-block布局

你可以使用 inline-block 来布局。有一些事情需要你牢记：

  • vertical-align属性会影响到inline-block元素，你可能会把它的值设为top
  • 你需要设置每一列的宽度
  • 如果HTML源代码中元素之间有空格，那么列与列之间会产生空隙

flexbox
