PSR编码规范

发表于 2017-03-19 |

本文内容来自于 Laravel-china 创始人 Summer

所有 PSR 规范请见：https://psr.phphub.org/

在这里为了自己方便查看，所以做了一些简单的总结。

关于「能愿动词」的使用

为了避免歧义，文档大量使用了「能愿动词」，对应的解释如下：

  • 必须(MUST) ：绝对，严格遵循，请照做，无条件遵守；

  • 一定不可(MUST NOT) ：禁令，严令禁止；

  • 应该(SHOULD) ：强烈建议这样做，但是不强求；
  • 不该(SHOULD NOT) ：强烈不建议这样做，但是不强求；
  • 可以(MAY) 和 可选 (OPTIONAL) ：选择性高一点，在这个文档内，此词语使用较少；

PSR-1 基本代码规范

  • PHP代码文件 必须 以 <?php 或 <?= 标签开始；

  • PHP代码文件 必须 以 不带 BOM 的 UTF-8 编码；

  • PHP代码中 应该 只定义类、函数、常量等声明，或其他会产生 副作用 的操作（如：生成文件输出以及修改 .ini 配置文件等），二者只能选其一；

  • 命名空间以及类 必须 符合 PSR 的自动加载规范：PSR-4 中的一个；

  • 类的命名 必须 遵循 StudlyCaps 大写开头的驼峰命名规范；

  • 类中的常量所有字母都 必须 大写，单词间用下划线分隔；

  • 方法名称 必须 符合 camelCase 式的小写开头驼峰命名规范。

“display”属性

display 是CSS中最重要的用于控制布局的属性。每个元素都有一个默认的 display 值，这与元素的类型有关。对于大多数元素它们的默认值通常是 block 或 inline 。一个 block 元素通常被叫做块级元素。一个 inline 元素通常被叫做行内元素。

block

div 是一个标准的块级元素。一个块级元素会新开始一行并且尽可能撑满容器。其他常用的块级元素包括 p 、 form 和HTML5中的新元素： header 、 footer 、 section 等等。

归并排序

发表于 2017-03-09 |

归并算法的基本思想

归并排序是建立在归并操作上的一种有效的排序算法。该算法是采用分治法（Divide and Conquer）的一个非常典型的应用。“归并”的含义是将两个或两个以上的有序表组合成一个新的有序表。因此归并排序就是利用归并的思想来实现的，假设初始序列含有n个记录，则可以看成是n个有序的子序列，每个子序列的长度为1，然后两两归并，得到 (n+1)/2个长度为2或1的有序子序列；再两两归并，….，如此重复，直至得到一个长度为n的有序序列位置，这种排序方法称为2路归并排序。

在mac上如何使用docker安装oracle 11g?

发表于 2017-03-06 |

1.在docker官网下载安装: docker for Mac

2.打开doker，配置国内加速器，例如：
Daocloud加速器
添加到advanced/registry mirrors

3.测试docker：

docker ps查看当前容器

docker run hello-world 出现hello，world则成功。

4.安装oracle 11g:

github地址：wnameless/oracle-xe-11g
docker pull wnameless/oracle-xe-11g

5.运行oracle:

首先运行oralce镜像,创建一个新的容器
docker run -d -p 49160:22 -p 49161:1521 wnameless/oracle-xe-11g

如果要远程连接数据库：
docker run -d -p 49160:22 -p 49161:1521 -e ORACLE_ALLOW_REMOTE=true wnameless/oracle-xe-11g

进入容器中
docker exec -it <name> bash (name= ContainerID)

创建完容器，每次打开及关闭容器命令：

Docker start <name>

Docker stop <name>

连接数据库命令：
sqlplus system/oracle

连接数据库的基本设置：

Password for SYS & SYSTEM

oracle

Login by SSH

Docker 常用命令：

docker info 查看docker信息

Docker ps 查看当前容器 后面加上-a查看所有容器

docker rm 删除容器

docker images 查看所有镜像

docker rmi 删除镜像
