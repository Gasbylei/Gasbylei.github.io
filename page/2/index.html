归并算法的基本思想

归并排序是建立在归并操作上的一种有效的排序算法。该算法是采用分治法（Divide and Conquer）的一个非常典型的应用。“归并”的含义是将两个或两个以上的有序表组合成一个新的有序表。因此归并排序就是利用归并的思想来实现的，假设初始序列含有n个记录，则可以看成是n个有序的子序列，每个子序列的长度为1，然后两两归并，得到 (n+1)/2个长度为2或1的有序子序列；再两两归并，….，如此重复，直至得到一个长度为n的有序序列位置，这种排序方法称为2路归并排序。

在mac上如何使用docker安装oracle 11g?

发表于 2017-03-06

1.在docker官网下载安装: docker for Mac

2.打开doker，配置国内加速器，例如：
Daocloud加速器
添加到advanced/registry mirrors

3.测试docker：

docker ps查看当前容器

docker run hello-world 出现hello，world则成功。

4.安装oracle 11g:

github地址：wnameless/oracle-xe-11g
docker pull wnameless/oracle-xe-11g

5.运行oracle:

首先运行oralce镜像,创建一个新的容器
docker run -d -p 49160:22 -p 49161:1521 wnameless/oracle-xe-11g

如果要远程连接数据库：
docker run -d -p 49160:22 -p 49161:1521 -e ORACLE_ALLOW_REMOTE=true wnameless/oracle-xe-11g

进入容器中
docker exec -it <name> bash (name= ContainerID)

创建完容器，每次打开及关闭容器命令：

Docker start <name>

Docker stop <name>

连接数据库命令：
sqlplus system/oracle

连接数据库的基本设置：

Password for SYS & SYSTEM

oracle

Login by SSH

Docker 常用命令：

docker info 查看docker信息

Docker ps 查看当前容器 后面加上-a查看所有容器

docker rm 删除容器

docker images 查看所有镜像

docker rmi 删除镜像
